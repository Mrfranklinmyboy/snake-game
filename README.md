#  Змейка

Игра "Змейка" на HTML, CSS и JavaScript с улучшенным дизайном и дополнительными возможностями.

## Технологии

- HTML5
- CSS3  
- JavaScript
- Canvas API

## Что умеет игра

- 3 типа еды (обычная, бонусная, скоростная)
- 3 уровня сложности
- Сохранение рекордов
- Управление с клавиатуры и свайпами
- Адаптивный дизайн для телефонов
- Система уровней и ускорение
