#  Змейка

Игра "Змейка" на HTML, CSS и JavaScript с улучшенным дизайном и дополнительными возможностями.

## Технологии

- HTML5
- CSS3  
- JavaScript
- Canvas API

## Как запустить 
1. Скачай файлы проекта
2. Открой файл `index.html` в браузере
3. Играй!
ссылка https://mrfranklinmyboy.github.io/snake-game/


## Что умеет игра

- 3 типа еды (обычная, бонусная, скоростная)
- 3 уровня сложности
- Сохранение рекордов
- Управление с клавиатуры и свайпами
- Адаптивный дизайн для телефонов
- Система уровней и ускорение
